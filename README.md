# Quick Search

Quick Search is a lightweight, keyboard-driven Firefox browser extension that lets you search, open, and manage websites instantly from a popup.

It is designed for speed, minimalism, and privacy â€” no tracking, no analytics, no external services.

---

##  Features

-  **Real-time name-based search**
-  **Press Enter to open the first result**
-  **Click any result to open it**
-  **Optional single-key shortcuts** (inside the popup)
-  **Edit mode** to add, update, or delete websites
-  **Dark mode** (follows system theme + manual toggle)
-  **All data stored locally** using browser storage
-  **Zero data collection**

---

## How It Works

1. Open the launcher using the toolbar icon or a keyboard shortcut.
2. Start typing the **website name**.
3. Press **Enter** to open the top result or click a result.
4. (Optional) Assign a **single-character shortcut** to open a site instantly.
5. Enable **Edit Mode** to manage saved websites.

---

## Keyboard Shortcuts

| Action | Key |
|------|----|
| Open launcher | `Ctrl + Shift + Q` (configurable) |
| Open first result | `Enter` |
| Open site by shortcut | Assigned key (popup only) |

> Note: Shortcut keys work **inside the popup only**, due to browser security restrictions.

---

## Edit Mode

When **Edit Mode** is enabled:
- Clicking a site loads it into the editor
- You can modify:
  - Name
  - URL
  - Shortcut key (optional)
- Delete removes the site permanently

Shortcut keys are validated to prevent duplicates.

---

## Dark Mode

- Automatically follows system/browser theme by default
- Manual toggle available in the popup
- Preference is saved locally

---

## Privacy

Quick Search does **not**:
- Collect personal data
- Collect technical data
- Track user behavior
- Communicate with external servers

All configuration is stored locally using the browser storage API.
